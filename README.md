# AI-OpenMic

Code and Dataset repository for the EMNLP 2021 paper:

[“So You Think You’re Funny?”: Rating the Humour Quotient in Standup Comedy](https://arxiv.org/pdf/2110.12765.pdf).

* Paper: [Link](https://arxiv.org/abs/2110.12765)
